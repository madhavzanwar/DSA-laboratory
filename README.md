## 📖 About

A structured collection of DSA lab assignments, daily practice programs, and problem-solving exercises implemented in **C++**, covering linked lists, trees, graphs, sorting, searching, hashing, and greedy algorithms — with time & space complexity analysis where applicable.

## ⚙️ Compile & Run

```bash
git clone https://github.com/madhavzanwar/DSA-laboratory.git
cd DSA-laboratory

g++ frequency.cpp -o frequency
./frequency
```

## 📈 Complexity Notes

Each solved assignment includes a brief time & space complexity discussion in code comments, e.g.:

```cpp
// Time Complexity: O(n log n) — dominated by sort
// Space Complexity: O(n) — auxiliary hash map
```
